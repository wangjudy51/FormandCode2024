Project 6: Optical Illusion

Design Process + Reflections

This is an optical illusion based on the idea that repetitive shapes can help calm others and make it seem like time has slowed down. I created this to mimic a meditation visual, in which it helps users pace their breathing. Using for loops helped to make the drawing process very quick. I also used a rotating plane in 3D space because it felt unexpected.
