Project 9: Video Capture

Design Process + Reflections

This camera and video manipulation project uses the createCapture and an array that repeatedly draws pixels. I struggled to write the code but by following a tutorial, I was able to figure out how get information from the camera's video as input to manipulate the color of the output for each pixel. I can see createCapture being used for fun camera filters, abstracted portraits, and also as a tool used to blur images/videos. It can help with things like obscuring your face from others.
