Project 8: Star Soup Game

Design Process + Reflections

This game is inspired by my love for cooking simulator games, 8-bit graphic styles, and astronomy. In this game, the main objective is to collect falling stars by moving the character, and making star soup. The player must catch 15 or more stars to sucessfully make star soup. I was not able to use the background and pre-made images I had planned to but I worked around it by using p5 to draw shapes. The most difficult parts of this project was figuring out where certain lines of code belong and when to call certain functions. Following example projects and keeping things as simple as possible helped me in completing this game.
