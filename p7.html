Project 7: Data Self Portrait

Design Process + Reflections

This data portrait sketch track the different food groups I have eaten over 5 days and the time at which I had eaten them. Each circle is colored to represent food groups eaten at a certain time. There is a total of 5 food groups, each with an assigned color. (carb = yellow, dairy = white, protein = red, vegetable = green, fruit = purple). The more muddy a circle is, the more different types of food I had = the more balanced the meal. In order to create this sketch, I had to simplify the data quite a bit in order to make the image more comprehensible. This created a good amount of context that would better help viewers understand my eating habits.
