Project 1: Lost in Translation

Description

This is a drawing I made using shape primitives with p5.

Design Process

I started with a 2D grid system I made in Illustrator and then wrote code for it.

Reflections

This exercise was enjoyable and not very difficult. It reminds me a lot of the Python turtle function with its shape-drawing functions using coordinates and hierarchy but has its differences which I am trying to get used to. I definitely feel that p5js is more efficient in drawing shapes compared to using Python's turtle functions.
